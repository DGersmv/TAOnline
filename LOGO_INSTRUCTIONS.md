# Инструкция по добавлению логотипа

## Как добавить логотип на экран входа

### Вариант 1: Использование файла изображения (рекомендуется)

1. **Подготовьте файл логотипа:**
   - Формат: PNG (рекомендуется) или WebP
   - Размер: минимум 512x512 пикселей для лучшего качества
   - Название файла: `logo.png` или `logo.webp`

2. **Поместите файл в папку:**
   ```
   app/src/main/res/drawable/logo.png
   ```
   или
   ```
   app/src/main/res/drawable/logo.webp
   ```

3. **Пересоберите проект** - логотип автоматически появится на экране входа

### Вариант 2: Использование разных размеров (для оптимизации)

Если хотите оптимизировать для разных плотностей экрана:

1. Подготовьте несколько размеров:
   - `logo.png` (базовая версия) → `app/src/main/res/drawable/logo.png`
   - `logo-hdpi.png` → `app/src/main/res/drawable-hdpi/logo.png`
   - `logo-xhdpi.png` → `app/src/main/res/drawable-xhdpi/logo.png`
   - `logo-xxhdpi.png` → `app/src/main/res/drawable-xxhdpi/logo.png`
   - `logo-xxxhdpi.png` → `app/src/main/res/drawable-xxxhdpi/logo.png`

2. Размеры для разных плотностей:
   - mdpi: 1x (базовый размер)
   - hdpi: 1.5x
   - xhdpi: 2x
   - xxhdpi: 3x
   - xxxhdpi: 4x

   Например, если базовый размер 200x200:
   - mdpi: 200x200
   - hdpi: 300x300
   - xhdpi: 400x400
   - xxhdpi: 600x600
   - xxxhdpi: 800x800

### Текущее поведение

- Если файл `logo.png` или `logo.webp` найден в `drawable` - отображается логотип
- Если файл не найден - отображается текст "ТАШИАНи" с подзаголовком "Вход в систему"

### Рекомендации

1. **Формат PNG** - лучше всего подходит для логотипов с прозрачностью
2. **Размер файла** - старайтесь оптимизировать (можно использовать инструменты сжатия)
3. **Прозрачный фон** - если логотип имеет прозрачный фон, используйте PNG
4. **Цвета** - логотип будет хорошо смотреться на светлом фоне экрана входа

### Проверка

После добавления файла:
1. Пересоберите проект (Build → Rebuild Project)
2. Запустите приложение
3. На экране входа должен появиться логотип

Если логотип не появляется:
- Проверьте, что файл называется точно `logo.png` или `logo.webp`
- Проверьте, что файл находится в `app/src/main/res/drawable/`
- Убедитесь, что файл не поврежден
- Проверьте логи Android Studio на наличие ошибок
